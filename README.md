# CSC1034 Assessment 3 : PageRank

## Brief Project Description:
Project 3 is about PageRank, a concept that refers to the algorithm Google uses to show us the most relevant
websites at the top of search results. To estimate PageRank, this project has two methods Stochastic and 
Distribution. Through the command lines, you can choose which algorithm to use, the number of repetitions 
or steps, graph representation type, among others.

## Prerequisites:
To see the prerequisites for running this script please see the [Requirements.txt](Requirements.txt)

## Testing its Features:

### PageRank Methods: -m / --method
* **Stochastic: Uses random walks to estimate PageRank**

    _(Default version uses 1000000 repetitions in an adjacency list and shows 20 top websites)_

    `python page_rank.py school_web2024-1.txt -m stochastic`

* **Distribution: Uses probability distributions to estimate PageRank**

    _(Default version uses 100 repetitions in an adjacency list and shows 20 top websites)_

    `python page_rank.py school_web2024-1.txt -m distribution`

### PageRank Methods: -r / --repeats (Stochastic)
**This is the number of repetitions for the random walks in the stochastic method.**
_(The default is 1,000,000)_

1. For 500 repetitions:
`python page_rank.py school_web2024-1.txt -m stochastic -r 500`
2. For 500000 repetitions:
`python page_rank.py school_web2024-1.txt -m stochastic -r 500000`

### PageRank Methods: -s / --steps (Distribution)
**This is the number of steps for the distribution method.**
_(The default is 100)_

1. For 500 steps:
`python page_rank.py school_web2024-1.txt -m distribution -s 500`
2. For 5000 steps:
`python page_rank.py school_web2024-1.txt -m distribution -s 5000`

### PageRank Methods: -n / --number 
Specifies the number of top results to display (default is 20).

1. (Stochastic) Display top 5 websites:
`python page_rank.py school_web2024-1.txt -m stochastic -n 5`
2. (Distribution) Display top 20 websites:
`python page_rank.py school_web2024-1.txt -m distribution -n 20`

### Graph Representations (Stochastic) --representation 
_All are using 10000 repetitions_
1. **Adjacency List:** `python page_rank.py school_web2024-1.txt -m stochastic --representation adj_list -r 10000`
2. **Adjacency Matrix:** `python page_rank.py school_web2024-1.txt -m stochastic --representation adj_matrix -r 10000`
3. **Edge List:** `python page_rank.py school_web2024-1.txt -m stochastic --representation edge_list -r 10000`
4. **Network X:** `python page_rank.py school_web2024-1.txt -m stochastic --representation networkx -r 10000`

### Graph Representations (Distribution) --representation 
_All are using 100 steps_
1. **Adjacency List:** `python page_rank.py school_web2024-1.txt -m distribution --representation adj_list -s 100`
2. **Adjacency Matrix:** `python page_rank.py school_web2024-1.txt -m distribution --representation adj_matrix -s 100`
3. **Edge List:** `python page_rank.py school_web2024-1.txt -m distribution --representation edge_list -s 100`
4. **Network X:** `python page_rank.py school_web2024-1.txt -m distribution --representation networkx -s 100`
